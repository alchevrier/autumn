# autumn-gradle-plugin

The Gradle integration hook for the `autumn-compiler-plugin`.

Applying `id("dev.autumn.plugin")` links the K2 IR generation extensions to the Kotlin compilation lifecycle. This ensures that the Circuit-Based boundaries—like strict Zero-Allocation matrices and constant-evaluated compiler budgets—are rigorously enforced on every `check` or `build` across all platforms (iOS, Android, JVM, and Web).
